# CONTRIBUTE

## Python Environment

1. Setup virtualenv 

```bash
# Create venv
python3 -m venv venv
# Activate venv
source venv/bin/activate
```

2. Install dependencies

```bash
(venv) % pip install -r requirements_dev.txt
```
